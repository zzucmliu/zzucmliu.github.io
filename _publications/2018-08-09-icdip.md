excerpt: 'With the rapid development of computer graphics technology and mobile terminal, Displaying effect on mobiles becomes more and more significant. In this paper,we implement a new interpolation method by using a kind of discontinuous B-spline on Android platform.This interpolation can truncate the interpolation curve at the place with a large data skip. We can keep the image edge sharp by applying it to image processing. This interpolation algorithm runs smoothly on Android mobile phone, which provides a new idea for the application and improvement of image processing algorithm effect on Android platform.'
